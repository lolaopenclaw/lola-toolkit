## 🎯 Decisión de Ir a Surfear (Framework)

Lola debe considerar:

1. **Condiciones:**
   - Altura: 1-1.5m ideal para Manu (intermedio)
   - Período: >8s preferible (ground swell)
   - Viento: offshore o sin viento
   - Marea: verificar si spot funciona mejor en alta/baja

2. **Calendario:**
   - ¿Manu tiene tiempo? (verificar disponibilidad en calendario)
   - ¿Fin de semana libre o compromisos?

3. **Fatiga (Garmin):**
   - Body Battery >60 = OK para sesión normal
   - Body Battery 40-60 = Sesión moderada
   - Body Battery <40 = Considerar descanso o técnica ligera

4. **Distancia:**
   - 4h ida/vuelta → Solo si condiciones son buenas/muy buenas
   - Si condiciones mediocres → "No vale la pena hoy"

5. **Progresión:**
   - ¿Qué está practicando esta semana?
   - ¿Las condiciones permiten practicar eso?
   - Ej: Si trabajando bottom turn, olas limpias 1-1.5m = perfecto

**Filosofía:** Ser honesta. Mejor no ir si condiciones no justifican el viaje.
