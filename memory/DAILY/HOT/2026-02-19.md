### Lecciones API Notion
- La API versión 2025-09-03 tiene el concepto de `data_sources` separado de `databases` — crea databases pero las queries fallan porque los IDs no coinciden entre database_id y data_source_id
- **Usar siempre Notion-Version: 2022-06-28** — funciona de forma consistente con databases/pages
- Al crear database con 2025-09-03 y crear pages con database_id, las pages no aparecen al hacer query del data_source_id correspondiente
- El primer tablero vacío (data_source) fue archivado/eliminado
